Ein Werkzeug für schnelle Produktbild-Iteration: dasselbe Produkt-Referenzbild,
viele verschiedene Prompts und mehrere Provider — parallel gerendert.

- **Kern-Workflow:** gleiches Produkt + viele Prompts + mehrere Modelle → parallele Varianten
- **Master-Prompt:** eine globale Konsistenz-Regel für alle Agenten eines Laufs
- **Multi-Provider:** OpenAI, Google, FLUX, fal, Replicate, Stability, Ideogram, Recraft, Leonardo
- **Drei Oberflächen aus einer Engine:** Desktop-App (.exe), CLI und passwortgeschützte Web-App
- Rate-limit-sicherer globaler Scheduler, Budget-Deckel, Live-Ergebnisse

<div data-lang="en">

A tool for fast product-image iteration: the same product reference image, many
different prompts and several providers — rendered in parallel.

- **Core workflow:** same product + many prompts + several models → parallel variants
- **Master prompt:** one global consistency rule for every agent in a run
- **Multi-provider:** OpenAI, Google, FLUX, fal, Replicate, Stability, Ideogram, Recraft, Leonardo
- **Three surfaces, one engine:** desktop app (.exe), CLI and a password-protected web app
- Rate-limit-safe global scheduler, budget caps, live results
