## Context

Berry ist die Collector-App für One-Piece-Karten auf dem Phone. Design v3 ersetzt Fantasy-Glow durch ein kühleres, präzises System – ohne die Marke zu verlieren.

## Process

1. Brief statt Skin-Tausch.
2. Obsidian-Edge Surfaces, klare Typo.
3. Vite → Capacitor → versionierte APKs.
4. Branding-Exploration parallel zum UI.

## Craft

Echte App-Screens: Home, Collection, Search. Tap-Ziele ≥ 44×44, Offline via Dexie, Design-System v3.

## Result

Shipbare Android-App – Design und Release aus einer Hand, parallel zu NEXUS.

## Learnings

Mobile braucht strengere Budgets als Desktop-Glass: weniger Frost, mehr Hierarchie.
